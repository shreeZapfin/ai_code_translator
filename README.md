->AI Code Explainer

AI Code Explainer is a simple web application that helps developers to understand the code quickly.  
Users can paste a code snippet, and the application uses an AI model to generate a explanation of the code.

->Features

- Paste any code snippet 
- AI generates a simple explanation
- UI
- Node.js backend with REST API
- Uses OpenAI API for code explanation

->Tech Stack

->Frontend:
- HTML
- CSS
- JavaScript

->Backend:
- Node.js
- Express.js

->API:
- OpenAI API
